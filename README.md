# 코드 봇

다양한 프로그래밍 언어의 코드를 실행할 수 있는 디스코드 봇

## 슬래시 명령어
/help  
도움말 표시 (자기 자신한테만 보임)  
  
/code  
코드 입력 모달 표시  

사용 가능 언어는  
https://tio.run/  
에서 확인 가능
